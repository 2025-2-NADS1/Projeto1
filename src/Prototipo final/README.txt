Resumo: 
MasterScore é um jogo de perguntas e respostas que cada pergunta 
recompensa com uma quantidade de pontos, estes pontos podem ser 
usados para comprar dicas de onde se encontrar cupons da PicMoney

Como executar:
Abra o arquivo >> MasterScore.sln<< no Visual Studio e altere a variavel do 
caminho para o banco de dados chamado >>caminhoDB<< para o 
arquivo >>MasterScore.db<<, que está dentro da pasta junto com o sln, e 
execute a solução
